# LLM-agents

This project explores two types of LLM agents:

- ReAct agent for deadline tracking
- LangGraph-based weather assistant using an external API

The goal of the project is to compare different paradigms of building LLM agents for practical user tasks.

## Motivation

This project was built as a practical exploration of LLM agents:
- ReAct for reasoning + acting with tools and memory
- LangGraph for structured workflows and API integration

## Project Structure

### 1. ReAct Deadline Assistant
An agent with tools and memory that helps track deadlines and manage tasks.

### 2. LangGraph Weather Assistant
A graph-based agent that processes user weather queries, calls an external API, and generates a human-readable response.
